# 🚀 Day 03 - Flexbox Architecture

## 📌 Overview

This project is part of the **50 Days Web Development Challenge** by Synexus.

Today focuses on mastering **CSS Flexbox**, the modern layout system used to build responsive user interfaces.

---

## 🎯 Objectives

- Learn Flexbox fundamentals
- Create a responsive navigation bar
- Align navigation links using Flexbox
- Build a perfectly centered Hero section
- Understand Main Axis and Cross Axis
- Use Flexbox instead of float layouts

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox)

---

## ✨ Features

✔ Flex Navigation Bar

✔ Logo aligned left

✔ Navigation links aligned right

✔ Hero Section centered horizontally and vertically

✔ Hover effects on navigation links

✔ CSS Variables

✔ Clean layout
