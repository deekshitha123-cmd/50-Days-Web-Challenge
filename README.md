# 🌐 Community Platform - Day 01

## 📌 Project Overview

This project is part of the **50 Days Web Development Challenge**.

The objective of Day 01 is to understand and implement **Semantic HTML5** by building a well-structured webpage without relying on generic container elements like `<div>` wherever semantic tags are available.

---

## 🎯 Objective

- Learn Semantic HTML5
- Understand webpage structure
- Build an accessible webpage
- Improve SEO using semantic elements
- Practice clean HTML architecture

## ♿ Accessibility

This project follows basic accessibility practices by using:

- `role="banner"`
- `role="navigation"`
- `role="main"`
- `role="contentinfo"`

along with semantic HTML elements.
