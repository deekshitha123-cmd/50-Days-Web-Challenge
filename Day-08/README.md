# 🚀 Day 08 – Native CSS Transitions & Micro-Interactions

## 📌 Overview

This project is part of the **50 Days Web Development Challenge by Synexus**.

The objective of Day 08 was to improve the user experience by implementing smooth and lightweight CSS micro-interactions using only native CSS—without any JavaScript.

The project enhances the Membership Application Form with modern hover effects, smooth focus transitions, and subtle animations to create a more interactive and professional interface.

---

## ✨ Features

- 🎨 Modern Membership Application Form
- 📱 Fully Responsive Design
- 🚀 Smooth Button Hover Animation
- 🎯 Lift Effect using CSS Transform
- 💡 Animated Form Input Focus States
- 🌟 Enhanced Box Shadow Effects
- ⚡ Hardware Accelerated Animations
- 🎭 Cubic-Bezier Transition for Smooth Motion
- 🚫 Disabled Button Styling
- 🧹 Clean and Organized CSS Structure
