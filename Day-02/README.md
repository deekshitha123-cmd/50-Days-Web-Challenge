## 📖 Key Learnings

- Every HTML element follows the **CSS Box Model**.
- **Padding** creates space inside an element.
- **Margin** creates space outside an element.
- Borders add visible boundaries around elements.
- Using `box-sizing: border-box` makes layouts easier to manage.
- Applying the Universal Reset removes default browser spacing for consistent designs.
