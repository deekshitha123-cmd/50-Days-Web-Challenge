# 🗓️ Day 06 – Responsive Engineering & Media Queries

## 📌 Overview

Today I transformed my website into a fully responsive web application using **CSS Media Queries**. Instead of keeping a fixed desktop layout, the website now adapts automatically to different screen sizes including desktops, tablets, and mobile phones.

This project demonstrates how responsive web design improves user experience by making layouts flexible and easy to use on any device.

---

## 📱 Responsive Breakpoints

### 🖥 Desktop
- Screen Width: **Above 900px**
- Grid Layout: **3 Columns**

### 💻 Tablet
- Screen Width: **900px and below**
- Grid Layout: **2 Columns**

### 📱 Mobile
- Screen Width: **600px and below**
- Grid Layout: **1 Column**
- Navigation changes to a vertical layout
