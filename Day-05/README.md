# 🗓️ Day 05 – CSS Grid Architecture (The Initiatives Gallery)

## 📌 Overview

On Day 05 of the **50 Days Web Development Challenge**, I explored **CSS Grid**, a powerful layout system for creating structured two-dimensional designs.

In this project, I built an **Initiatives Gallery** consisting of reusable project cards arranged in a responsive grid layout. This challenge helped me understand how CSS Grid simplifies building clean and scalable user interfaces.

---

## 🎯 Objectives

- Learn the fundamentals of CSS Grid.
- Create a responsive 2D layout.
- Build reusable initiative cards.
- Understand fractional units (`fr`) and the `gap` property.
- Practice modern frontend layout techniques.

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## 📚 Concepts Learned

- `display: grid`
- `grid-template-columns`
- `repeat()`
- Fractional Units (`fr`)
- `gap`
- `grid-column: span`
- Card-based UI Design
- Responsive Layout Basics

---

## ✨ Features

- Clean and structured Initiatives Gallery
- Three responsive project cards
- Featured card spanning two columns
- Consistent spacing using CSS Grid
- Modern card styling with shadows and rounded corners
