# 🌐 Day 09 – Accessibility (a11y) Standards & Inclusive Engineering

## 📌 Project Overview

This project is part of the **50 Days Web Development Challenge by Synexus**.

The objective of Day 09 was to improve the website's accessibility by following **Web Content Accessibility Guidelines (WCAG)**. The focus was on making the website usable for everyone, including people who navigate using keyboards, screen readers, and assistive technologies.

---

## 🚀 Features

- 🔹 Accessible Navigation Bar
- 🔹 Skip to Main Content Link
- 🔹 Responsive Hero Section
- 🔹 Interactive Team Profile Cards
- 🔹 Membership Application Form
- 🔹 Responsive Footer
- 🔹 Glassmorphism UI Design
- 🔹 Smooth CSS Transitions & Hover Effects

---

## ♿ Accessibility Improvements

- ✅ Skip to Main Content link for keyboard users
- ✅ Semantic HTML using `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>`
- ✅ ARIA Labels for navigation, buttons, form fields, and social media icons
- ✅ Meaningful `alt` text for all profile images
- ✅ Keyboard-friendly navigation using `:focus-visible`
- ✅ Improved color contrast for better readability
- ✅ Visible focus indicators for interactive elements
- ✅ Screen-reader-friendly form labels
